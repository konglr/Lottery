# 排列三 (PL3) 官方规则

### 1. 游戏规则 (Game Rules)
*   **号码范围**：从 000 到 999 之间的三位数字。
*   **投注方式**：
    *   **直选**：数字、顺序完全一致。
    *   **组选三**：两个数字相同，不限顺序。
    *   **组选六**：三个数字不同，不限顺序。

### 2. 开奖与规则 (Draw Info)
*   **中奖号码**：采用体育彩票“排列5”当期开奖号码的前三位数字。
*   **开奖时间**：每天 20:25 开奖。

### 3. 固定奖金 (Fixed Prizes)
*   **直选**：单注奖金 **1040** 元。
*   **组选三**：单注奖金 **346** 元。
*   **组选六**：单注奖金 **173** 元。

### 4. 销售限制
*   排列三对每期全部投注号码实行限量销售，若触及风险管控阈值，则可能暂停该号码的投注。
