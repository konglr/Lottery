# 中国彩票官方规则手册

## 常用通用规则

1. **购彩限制**：禁止向未成年人销售彩票或兑付奖金。
2. **兑奖期限**：自开奖之日起 60 个自然日内，逾期未兑视为弃奖（特殊情况除外）。
3. **有效凭证**：中奖彩票是唯一的兑奖凭证，请妥善保管。
4. **单注上限**：浮动奖单注最高限额通常为 500 万或 1000 万（视彩种及调节基金情况而定）。
5. **投注倍数**：单张彩票多倍投注通常限制在 2-99 倍。
