# 福彩3D 官方规则

### 1. 游戏规则 (Game Rules)
*   **号码范围**：从 000 至 999 之间的三位数字。
*   **投注模式**：
    *   **单选**：号码及位置完全一致。
    *   **组选三**：三位数中有两个数字相同（如122），不限位置。
    *   **组选六**：三个数字各不相同（如123），不限位置。

### 2. 开奖方式 (Draw Method)
*   **开奖时间**：每天 21:15 开奖。
*   **方式**：通过专用摇奖机摇出三个 0-9 之间的数字。

### 3. 中奖奖金 (Winnings)
*   **单选**：单注奖金固定为 **1040** 元。
*   **组选三**：单注奖金固定为 **346** 元。
*   **组选六**：单注奖金固定为 **173** 元。

### 4. 兑奖流程
*   兑奖期限为 60 个自然日。
*   单注中奖金额在一万元以上者，需按规定缴纳个人所得税（通常针对多倍投注情况）。
